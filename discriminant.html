Дискримина́нт многочлена — математическое понятие , обозначаемое буквами D или Δ


Дискриминант вычисляется по формуле D = b^2 — 4ac, где a, b и c — коэффициенты квадратного уравнения. Если D > 0, то уравнение имеет два корня; если D = 0, то уравнение имеет один корень; если D < 0, то уравнение не имеет действительных корней.


Если D > 0; то у уравнения 2 корня.

x1 = -b - √D /2a
x2 = -b + √D /2a

Если D = 0; то у уравнения 1 корень т.е первый и второй корень равны.


x1 = x2 =

-b/2a


Если D < 0; то у уравнения нету точных корней.
